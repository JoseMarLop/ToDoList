## Requisitos
- Node.js vXX
- MongoDB

## Pasos
1. Clona el repositorio:
   \`\`\`bash
   git clone <url-del-repo>
   \`\`\`

2. Instala dependencias:
   \`\`\`bash
   npm install
   \`\`\`

3. Crea un archivo \`.env\` con las siguientes variables:
   \`\`\`env
   PORT=3000
   DB_URI=mongodb://localhost:27017/tu_basedatos
   \`\`\`

4. Ejecuta el proyecto:
   \`\`\`bash
   npm run dev
   \`\`\`
